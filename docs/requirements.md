# Gony-Labs 프로젝트 요구사항 정의서 (v1.0)

## 1. 프로젝트 개요
- **목적**: 1인 개발자의 실무 역량과 AI 기술력을 증명하는 풀스택 포트폴리오 구축
- **핵심 가치**: 유지보수성, 확장성, 지능형 사용자 경험 (AI)

## 2. 핵심 기능 요구사항 (MVP)
### A. 프론트엔드 (Next.js)
- 사용자는 세련된 랜딩 페이지를 통해 개발자 'Gony'의 역량을 확인할 수 있어야 한다.
- 모든 페이지는 모바일과 데스크탑에서 최적화된 UI를 제공해야 한다. (PWA 고려)

### B. AI 서비스 (Python)
- 사용자는 AI 챗봇과 대화하여 질문을 주고받을 수 있어야 한다.
- 챗봇은 자연스러운 응답을 제공하며, 필요시 상품 정보를 추천해야 한다.

### C. 백엔드 및 데이터 (NestJS & DB)
- 사용자의 대화 기록과 기본 정보가 안전하게 데이터베이스에 저장되어야 한다.
- 상품 목록과 결제 상태를 관리하는 API가 정상 작동해야 한다.

## 3. 기술적 제약 사항
- Frontend: Next.js (App Router)
- Backend: NestJS
- AI: Python (LangChain 기반 예정)
- DB: PostgreSQL (Supabase 활용 권장)
